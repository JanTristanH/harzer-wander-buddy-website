# harzer-wander-buddy-website
Harzer Wander Buddy Website


## Used jeykell theme
[Memoirs](https://bootstrapstarter.com/jekyll-theme-memoirs/)

## Commands to run
- `gem install bundler`
- `bundle install`
- `bundle exec jekyll serve --watch`
